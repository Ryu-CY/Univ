B677007 류창열 

기본적인 동작은 과제 양식과 같습니다.
1. r, g, b를 누르면 각각 배경이 붉은색, 초록색, 파란색으로 바뀝니다.
2. 화면에 화면상 좌측부터 오각형, 사각형, 육각형을 만들고 각각 다른 축을 기준으로 회전시켰습니다.
	(축 기준이 (0, 0)지점을 기준으로 하여 제자리 회전이 아닌 크게 회전을 합니다.)
3. w, d를 누르면 각각 삼각형들로 이루어진 도형들이 삼각형 모양 테두리들로 이루어지게 되거나 꽉 찬 형태로 바뀝니다.
4. 1, 2를 누르면 각각 최대 밝기, 중간 밝기로 바뀝니다.

인클루드 부분과 라이브러리 부분 혹은 문자집합 등을 맞춰주어야 정상작동 합니다.

.ps파일을 두 개 이용하였기에 둘 다 같은 폴더 안에 있어야 합니다.
